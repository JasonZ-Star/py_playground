# 数据集说明

## boston_housing.csv

波士顿房价数据集的样本数据（20 行）。

### 列说明

| 列名 | 说明 | 单位 |
|------|------|------|
| CRIM | 城镇人均犯罪率 | - |
| ZN | 住宅用地所占比例 | % |
| INDUS | 城镇非零售商用土地的比例 | % |
| CHAS | 查尔斯河虚拟变量（1 = 邻近河流） | 0/1 |
| NOX | 一氧化氮浓度 | ppm |
| RM | 住宅平均房间数 | 间 |
| AGE | 1940 年之前建成的自用房屋比例 | % |
| DIS | 到波士顿五个中心区域的加权距离 | - |
| RAD | 辐射性公路的靠近程度指数 | - |
| TAX | 每 $10,000 全值财产税率 | $ |
| PTRATIO | 城镇师生比例 | - |
| B | 1000(Bk - 0.63)^2，Bk 是城镇中黑人的比例 | - |
| LSTAT | 人口中地位低下者的比例 | % |
| MEDV | 自住房的平均值（目标变量） | $1000 |

### 数据来源

本数据集是波士顿房价数据集的一个小样本，用于演示目的。完整数据集包含 506 行。

### 用法示例

```python
import pandas as pd

# 读取数据
df = pd.read_csv("/data/boston_housing.csv")

# 查看基本信息
print(df.head())
print(df.describe())

# 简单分析
print(f"平均房价: ${df['MEDV'].mean():.2f}k")
print(f"平均房间数: {df['RM'].mean():.2f}")
```
